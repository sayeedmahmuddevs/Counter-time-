# Stopwatch / Timer App ⏱️

A simple **Stopwatch/Timer Application** built with **HTML, CSS, and JavaScript**.  
It allows you to **start, stop, and reset** the timer with milliseconds precision.  

---

## Features

- Start the timer  
- Stop/pause the timer  
- Reset the timer  
- Shows **hours, minutes, seconds, and hundredths of a second**  
- Lightweight and easy to use  

---

## Demo

Open the `index.html` file in your browser to run the stopwatch.  

---

## Installation

## Usage

- Start: Click the Start button to begin counting.
- Stop: Click the Stop button to pause counting.
- Clear: Click the Clear button to reset the timer to 00:00:00:00.
